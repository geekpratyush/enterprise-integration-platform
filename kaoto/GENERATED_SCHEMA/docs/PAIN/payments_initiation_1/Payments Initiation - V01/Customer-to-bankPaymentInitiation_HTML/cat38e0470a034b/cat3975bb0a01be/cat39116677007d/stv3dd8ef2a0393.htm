Activity Recycle payment orders

Parent State/ActivityNone  


Documentation
The initiating party has received a payment status report where invalid/unsuccessful payment order have been rejected by the first or forwarding agent (for either technical or business reason or at the request of the initiating party itself-). No repair is possible on these payment orders. Consequently, there is a need to recycle, i.e. prepare the payment orders again, using different and correct information, the payment order and restart the submission to the bank.
