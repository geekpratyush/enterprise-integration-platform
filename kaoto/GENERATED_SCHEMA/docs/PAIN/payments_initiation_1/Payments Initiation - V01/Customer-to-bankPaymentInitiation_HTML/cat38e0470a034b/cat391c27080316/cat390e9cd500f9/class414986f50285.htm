Documentation
Occurs when the instructing party requests to modify a previously instructed payment.

This case is initially assigned by an instructing party to an instructed party. The assigned case can be forwarded through the payment chain.

The potential solution to such a case is:

- the modification request is accepted.
- the modification request is rejected. This happens if the instruction is already executed.
