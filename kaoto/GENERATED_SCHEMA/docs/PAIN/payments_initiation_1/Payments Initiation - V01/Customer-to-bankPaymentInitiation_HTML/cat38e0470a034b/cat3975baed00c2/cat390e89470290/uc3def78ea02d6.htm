<<business use case>> Assess interbank payment risk

Documentation
Definition : the settlement agent undertakes the necessary risk checks in order to process the payment further, for examples : embargo, sufficient credit, cover received, system limits.

Pre-condition : the settlement agent has received the payment order

Trigger : receipt of the payment order.

Post condition : the payment has been accepted or rejected.
