<<Amount>> DatedCurrencyAnd13DecimalAmount {Analysis}

Documentation
A number of monetary units specified in a dated currency where the unit of currency is explicit and compliant with ISO 4217. The number of fractional digits (or minor unit of currency) is not checked as per ISO 4217: It must be lesser than or equal to 13.
Note: The decimal separator is a dot.
