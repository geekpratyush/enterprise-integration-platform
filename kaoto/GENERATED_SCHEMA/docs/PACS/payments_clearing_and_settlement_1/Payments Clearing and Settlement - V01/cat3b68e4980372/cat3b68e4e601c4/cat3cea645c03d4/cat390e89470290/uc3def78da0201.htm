<<BusinessProcess>> Validate interbank payment

Documentation
Definition : the clearing agent undertakes the necessary content checks in order to process the payment further. For examples : authenticate the remitting party, validate syntax, confirm the SLA's.


Pre-condition : the clearing agent has received the payment order

Trigger : receipt of the payment order.

Post-condition : payment has been accepted or rejected. If accepted, the instructed payment is ready for the risk assessment or is executable.
